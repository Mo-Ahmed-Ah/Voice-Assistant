## About the Project
يهدف هذا المشروع إلى تحويل النصوص المكتوبة إلى صوت باستخدام مكتبة `pyttsx3` في بايثون. يمكن استخدام هذا المشروع لبناء تطبيقات المساعدة الصوتية أو التطبيقات التي تتطلب التفاعل الصوتي.

## Installation

لتثبيت المكتبات المطلوبة، تأكد من أن لديك Python مثبتًا على جهازك، ثم استخدم الأوامر التالية:

```bash
# تثبيت مكتبة pyttsx3
pip install pyttsx3
```
